A comprehensive web application for managing school operations, staff, news, and communications for School â„–30 in Ussuriysk.

## Features

- **Staff Management**: Detailed profiles of teachers and staff members
- **News System**: Categorized news and announcements
- **Vacancy Board**: Current job openings at the school
- **Document Repository**: Important school documents
- **Sponsor Information**: Details about school sponsors
- **Feedback System**: Contact form for parents and visitors
- **Responsive Design**: Mobile-friendly interface

## School Sections

### Main Navigation
- About the School
- News & Announcements
- Staff Directory
- Contact Information
- Job Vacancies
- Sponsors
- Documents
- Feedback Form

## Technologies Used

- **Backend**: Django 5.2
- **Frontend**: HTML, CSS
- **Database**: SQLite (can be configured for PostgreSQL/MySQL)
- **Additional Packages**:
  - django-filters for news categorization
  - Pillow for image handling
